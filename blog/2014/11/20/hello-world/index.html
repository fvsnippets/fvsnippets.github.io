Hello World

Nov 20th, 2014 3:58 pm

A few days ago, I added search functionality to Phindee so users can quickly find information about a particular happy hour. Search that is well-done often comes with autocomplete, and Phindee is no exception.

Autocomplete in Phindee does a couple of things for the user: 1) it reduces typing, 2) it lets the user quickly know if a specific happy hour is in the database, 3) it allows her to quickly find a particular happy hour that is in the database, and 4) it lets him know if the happy hour is currently happening because it will have a green circle next to it if thats the case.
